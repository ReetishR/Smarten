# SMARTEN - Invisible Anti-Counterfeiting Technology

This is a modern, professional website designed for VC presentations, showcasing the SMARTEN anti-counterfeiting solution.

## Tech Stack

- **Framework:** React + Vite (TypeScript)
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **Animations:** Framer Motion
- **Fonts:** Inter (Sans), Playfair Display (Serif)

## Getting Started

1.  **Install Dependencies:**
    ```bash
    npm install
    ```

2.  **Run Development Server:**
    ```bash
    npm run dev
    ```

3.  **Build for Production:**
    ```bash
    npm run build
    ```

## Design Philosophy

- **Visual Tone:** Sleek, intelligent, visionary.
- **Color Palette:** Deep Navy (#0f172a), Gold (#d4af37), Teal (#14b8a6), Ivory.
- **Structure:** Pitch-deck style flow (Problem -> Solution -> Market -> Competition -> Strategy).
